# DIY Quadcopter

A from-scratch DIY quadcopter project with custom PCB design using [KiCad](https://www.kicad.org/).

## Overview

This repository contains the hardware design files for a custom-built quadcopter. The goal is to design and build a drone from the ground up — including custom PCBs for the flight controller and associated electronics — rather than relying on off-the-shelf flight controller boards.

## Project Structure

```
Quadcopter/
├── LICENSE
├── README.md
└── (KiCad project files — schematics, PCB layouts, BOM)
```

> This project is in active development. Design files will be added as the project progresses.

## Tools & Technologies

- **[KiCad](https://www.kicad.org/)** — open-source EDA suite for schematic capture and PCB layout

## Getting Started

1. Install [KiCad 7.x or later](https://www.kicad.org/download/)
2. Clone this repository:
   ```bash
   git clone https://github.com/alecv2211/Quadcopter.git
   ```
3. Open the `.kicad_pro` project file in KiCad
